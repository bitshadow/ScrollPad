ScrollPad
=========

Scrollpad is Chromium add-on to provide an interface for scrolling webpages easy way.

###To Install:
1) Download zip or git clone.
2) Go To chrome -> extensions.
3) Use Developers Mode.
4) Load Unpacked extension.

Enjoy!
